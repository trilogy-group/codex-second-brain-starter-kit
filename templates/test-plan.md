# Test Plan Template

Use this as the acceptance checklist for the first product run.

## Must work

- manifest validates
- vault scaffolds successfully
- audit runs successfully
- readiness report generates successfully
- at least one real repository is connected
- at least one raw source is preserved in the vault
- at least one documentation-to-code path exists

## Nice to have

- a conflict note
- a local runtime note
- a bug candidate note
- the automation pack installed
